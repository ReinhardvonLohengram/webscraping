# webscraping
爬取有关“三山五园地区”政策信息

这个课题是为了做社会实践凑数据用的

html.py记录了需要爬取的内容

main.py将爬取各个政策公开页面

number.py记录了各个部门的网址信息

test.py则将爬取的各个表格中的内容整合为总表

我们将总表中的内容使用WPS的ai分析工具进行了处理
